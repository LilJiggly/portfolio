Spooky Products

Visual Interface Design / Filter system / Working on Grid

Tijdens de Minor Visual Interface Design kreeg ik de opdracht om alles wat ik geleerd had nog één laatste keer toe te passen tijdens de "Meesterproef". Mijn focus lag hierbij op het ontwikkelen van een website die de essentie van NieR: Automata - een intrigerende videogame - weerspiegelt. Mijn benadering was gericht op het implementeren van game design principes om de website een speelse en interactieve dimensie te geven die nauw aansluit bij het thema van de game.

Tijdens het creatieve proces heb ik mijn vaardigheden op het gebied van Visual Interface Design aangescherpt, waarbij ik elementen van de game wereldsubtiel en doordacht heb geïntegreerd. Het gebruik van Adobe XD stelde me in staat om een naadloze en boeiende gebruikerservaring te creëren.
